# Product Overview

cc-mirror Multi-Agent Orchestration 評価プロジェクト。Claude Code のベータ機能であるマルチエージェントオーケストレーションを、学術ベンチマーク（MARBLE）を使用して定量的に評価するための実験環境。

## Core Capabilities

1. **ベンチマーク実行**: MARBLE (MultiAgentBench) を使用したマルチエージェント評価
2. **評価レポート生成**: 定量的・定性的な評価結果のドキュメント化
3. **再現性確保**: 評価手順と環境設定の記録

## Target Use Cases

- Claude Code マルチエージェント機能の性能評価
- 複数エージェント協調タスクにおけるベンチマーキング
- 評価結果に基づく知見の共有

## Value Proposition

- cc-mirror 経由で有効化されるベータ機能の客観的評価
- ACL 2025 発表の学術ベンチマークによる信頼性の高い測定
- オープンソースによる評価手法の再現性
